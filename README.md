# xPCS Portfolio - Senior System Administrator | DevOps | AWS

Welcome to my professional portfolio website, hosted on GitHub Pages. This site showcases my expertise in cloud infrastructure, automation, and system administration.

## 🚀 Features

- **Modern & Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.
- **Glassmorphism UI**: Stylish and clean interface with subtle transparency effects.
- **Multi-section Layout**:
  - **Hero**: Introduction with dynamic background blobs.
  - **About**: Personal and professional summary.
  - **Skills**: Organized technical expertise across Cloud, DevOps, and SysAdmin.
  - **Projects**: Highlights of key professional and personal work.
  - **Experience**: A professional timeline of work history.
  - **Contact**: Direct links and call-to-action for networking.
- **Smooth Navigation**: Single-page application (SPA) style scrolling for a seamless user experience.

## 🛠️ Built With

- **HTML5**: Modern semantic structure.
- **CSS3**: Custom properties (variables), Grid, Flexbox, and keyframe animations.
- **JavaScript**: Minimal, performance-focused smooth scrolling logic.
- **Inter Font**: High-quality typography via Google Fonts.
